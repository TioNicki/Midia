# **App Name**: FaithFlow

## Core Features:

- Authentication: Secure user authentication with login functionality.
- Dashboard: Centralized dashboard with a sidebar menu for easy navigation.
- Duty Roster Management: Create, edit, and visualize duty rosters by date (admin only).
- Praise Song Database: Add and organize praise songs by worship service (admin only).
- Important Dates: Add and manage important events and dates (admin only).
- Feedback Submission: Form for members to submit feedback, ideas, or compliments.
- Access Control: Differentiate permissions between administrators (edit/delete) and regular members (view/submit feedback).

## Style Guidelines:

- Primary color: Deep Purple (#673AB7), evoking reverence and spirituality.
- Background color: Light Gray (#F5F5F5), creating a calm, uncluttered feel.
- Accent color: Teal (#009688), to highlight important actions and links.
- Body and headline font: 'PT Sans', a humanist sans-serif providing a modern and slightly warm feel.
- Use clear, simple icons to represent menu items and actions, following Material Design guidelines.
- Clean, responsive layout with a sidebar menu and well-defined sections for each feature.
- Subtle transitions and animations to enhance user experience without being distracting.
